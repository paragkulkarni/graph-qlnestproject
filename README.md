

## Description

### NestJs project for book api using graphql with apollo server for following operation
### We follo the code first approach to build graphql API
#### show data
#### add data
#### update data
#### delete data
#### find data by id

### We build following features of graphql
#### Query
#### Mutation
#### Ressolver
#### Input Arguments


## Project Url
localhost:3000/graphql


## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
